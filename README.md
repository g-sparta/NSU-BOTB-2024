# Oddball:The new normal
Submission for the 2024 HBCU Battle of the Brains.

## Description

Oddball: the new normal, is an add-on feature to the existing NFL+ subscription platform that incorporates a points reward system to drive subscriber engagement. Points can be accumulated through watching existing NFL+ content, particapting in AI/ML driven prediction polls, and partaking in different wellness activities.
